## ğŸš€ Getting Started

### Prerequisites

- Node.js (version 18.0 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/RimshaAslam23012001/Robotics_book.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Robotics_book
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm start
   ```

5. Open your browser to [http://localhost:3000](http://localhost:3000) to view the site.

### Available Scripts

- `npm start` - Starts the development server
- `npm build` - Builds the static site for production
- `npm serve` - Serves the built site locally
- `npm deploy` - Deploys the site (if configured)
